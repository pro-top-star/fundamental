Badge and Label

Badges and labels are used to indicate status. Colors, generally in combination with text, are used to easily highlight the state of an object.

- Black is used to indicate default or inactive status

- Green indicates positive status, used for active, published, approved

- Orange can be used for a warning status or to indicate that an action can be taken

- Red indicates error status
